# Data Visualization Module

The Data Visualization module provides advanced spatial and temporal analysis of epidemiological data.

## 🌟 Features
- **Spatial Analysis**: Hexagonal binning and cluster analysis of case coordinates.
- **Temporal Trends**: Visualize disease spread over time using dynamic charts.
- **Heat Maps**: Density visualization for identifying regional burdens.
- **Interactive Legends**: Dynamic filtering and data highlighting.

## 🛠️ Usage
1. Select the **Data Visualization** module from the dashboard.
2. Toggle between different visualization modes (Hexbin, Clusters, Heatmap).
3. Use the timeline slider to visualize temporal shifts.
