Batch Gradient Descent

The network learns by iteratively adjusting all weights to minimise the loss. Each epoch is one full pass through the training set:

For each epoch: 1. Forward pass → compute ŷ 2. Compute Loss (BCE) 3. Backprop → compute ∂L/∂w 4. Update: w ← w − η·∂L/∂w Loss should decrease monotonically (if learning rate is appropriate)

Batch Size & Dropout

Batch size controls how many patients are used per weight update. Smaller batches → noisy but faster convergence. Larger → smooth but slower.

Dropout randomly silences neurons during training — prevents the network from memorising individual patient noise.

Dropout(p=0.5): Each neuron disabled independently with probability p during training. → Forces redundant representations → Acts as ensemble of sub-networks

⚖️ Ethical Reality

Deep networks trained on ImageNet-like medical imaging datasets often perform worse on patients from hospitals with different scanner hardware. The model learns the scanner's artefacts, not the pathology. External validation across institutions is mandatory before clinical deployment.
