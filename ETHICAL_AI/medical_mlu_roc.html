How the ROC Curve Is Built

For every possible threshold (0→1), we compute TPR (Sensitivity) and FPR (1 − Specificity):

TPR = TP / (TP + FN) → of all diabetics, how many caught? FPR = FP / (FP + TN) → of all healthy, how many falsely flagged? Plot (FPR, TPR) for each threshold → ROC curve

A perfect classifier hugs the top-left corner. A random classifier lies on the diagonal.

Area Under the Curve (AUC)

AUC = probability that the model ranks a random positive higher than a random negative.

AUC = 1.0 → perfect model AUC = 0.5 → random guessing AUC = 0.0 → perfectly inverted model AUC is threshold-independent — a single number summarising overall discrimination.

Youden's J = TPR − FPR. The optimal clinical threshold maximises J.

⚖️ Ethical Reality

AUC treats all thresholds equally. In cancer screening, missing a malignancy (FN) is far worse than a false alarm (FP). The optimal threshold must reflect the cost asymmetry of clinical errors — not just mathematical balance.
