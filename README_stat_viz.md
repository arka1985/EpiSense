# Statistical Visualization Module

Interactive tools for exploring statistical distributions and fundamental concepts of data science.

## 🌟 Features
- **Distribution Exploration**: Visualize Normal, Binomial, and Poisson distributions.
- **Skewness & Kurtosis**: Real-time manipulation of distribution shapes.
- **Central Limit Theorem**: Dynamic simulations showing how means converge to normal distribution.
- **Interactive Widgets**: Adjust mean, standard deviation, and sample sizes to see visual impact.

## 🛠️ Usage
1. Open the **Statistical Visualization** module.
2. Select a distribution or concept tab.
3. Use the sliders to observe real-time changes in the probability density functions.
