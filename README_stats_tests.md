# Statistical Tests Module

A comprehensive automated environment for performing common biometric and epidemiological statistical tests.

## 🌟 Features
- **Parametric Tests**: T-Tests (Independent, Paired), ANOVA.
- **Non-Parametric Tests**: Mann-Whitney U, Wilcoxon Signed-Rank, Kruskal-Wallis.
- **Categorical Analysis**: Chi-Square tests for independence.
- **Normality Testing**: Shapiro-Wilk and visualization for assumption checking.
- **Automated Reporting**: Generates test summaries, p-values, and effect sizes.

## 🛠️ Usage
1. Navigate to **Statistical Tests**.
2. Select the appropriate test for your data type.
3. Input your data or upload a CSV file.
4. Review the automated interpretation of the $p$-value.
