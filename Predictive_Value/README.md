# Calculation of Positive and Negative Predictive Values of a Screening Test

This web application simulates the Positive Predictive Value (PPV) and Negative Predictive Value (NPV) of a screening test based on ranges of Sensitivity, Specificity, and Disease Prevalence using a Monte Carlo Simulation approach.

## Features

*   **Monte Carlo Simulation**: Run simulations (default 1000) to estimate PPV and NPV.
*   **Interactive Controls**: Dual-thumb sliders to set ranges for input parameters.
*   **Visualizations**: Interactive histograms and distribution density curves (KDE) for PPV and NPV.
*   **Result Interpretation**: Automatic medical interpretation of the results based on the inputs.
*   **Responsive Design**: Premium Neon/Dark-mode aesthetic.

## Technologies

*   HTML5
*   CSS3 (Custom Neon Design)
*   JavaScript (ES6+)
*   [Plotly.js](https://plotly.com/javascript/) for charting.
