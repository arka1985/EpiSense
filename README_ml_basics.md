# Machine Learning Basics Module

An educational exploration of machine learning concepts applied to epidemiological data.

## 🌟 Features
- **Linear Regression**: Visualize trends and make simple predictions for future case loads.
- **Correlation Analysis**: Identify relationships between different variables.
- **Model Training Simulation**: Interactive visualization of how weights shift during training.
- **Interactive Explanations**: Deep dives into loss functions, gradients, and overfitting.

## 🛠️ Usage
1. Choose the **ML Basics** module.
2. Adjust model complexity and observe changes in the fit line.
3. Review the educational cards for concept explanations.
