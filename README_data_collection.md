# GIS & Data Collection Module

The GIS (Geographic Information System) module allows users to track and visualize infectious disease cases on an interactive map.

## 🌟 Features
- **Real-time Case Entry**: Manually add cases with coordinates and disease types.
- **Hotspot Detection**: Automatic identification of high-density case clusters.
- **Disease Filtering**: Filter map visualization by specific infectious diseases.
- **CSV Data Sync**: Download current case data as CSV for offline analysis.

## 🛠️ Usage
1. Navigate to the **GIS & Case Mapping** section.
2. Use the "Add Simulated Case" button for random data or use the manual entry form.
3. Observe the dynamic heat map and hotspot markers.
