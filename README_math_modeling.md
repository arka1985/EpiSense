# Mathematical Modeling Module

A robust simulator for infectious disease epidemiology using deterministic and stochastic models.

## 🌟 Features
- **Deterministic Models**: SIR, SEIR, and SIS simulations using customizable differential equations.
- **Stochastic Network Model**: Simulate spread through social contact networks with adjustable contact density ($k$).
- **Interactive Parameters**: Real-time adjustment of $\beta$, $\gamma$, $\sigma$, and transmission probabilities.
- **Outbreak Metrics**: Automatic calculation of $R_0$, Peak Infection Day, and Epidemic Duration.
- **MathJax Integration**: Live rendering of mathematical formulas for each model.
- **CSV Overlay**: Upload empirical data to compare real-world observations with theoretical curves.

## 🛠️ Models
- **SIR**: Susceptible-Infected-Recovered.
- **SEIR**: Adds an 'Exposed' state for incubation periods.
- **SIS**: For diseases without lasting immunity.
- **Network**: Mean-field approximation of network spreading.
