    📊 About This App

    NutriSense India provides comprehensive nutritional data for 542 Indian foods based on the Indian Food Composition Tables 2017 (IFCT 2017).


    Data source: IFCT 2017 by T. Longvah, R. Ananthan, K. Bhaskarachary and K. Venkaiah, published by the National Institute of Nutrition (NIN), Indian Council of Medical Research (ICMR).


    Code data from nodef/ifct2017 GitHub repository.

    📋 Nutrient Reference Values

    Daily reference values are based on adult RDA (Recommended Dietary Allowances) from international guidelines including WHO, US FDA, and ICMR standards for a 2000 kcal diet.
